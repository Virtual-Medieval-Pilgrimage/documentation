# Creating Additional Pilgrimages

You can make new pilgrimages by copying one of the pages in the _items folder.

1. Open the file you want to copy and use the "Raw" button to load it.
1. Select-all and copy the contents of the file.
1. Create a new file in the `_items` folder by finding the "Add a File" button.
1. Paste the contents of the file you copied into this new file and update the necessary elements of the new file.
